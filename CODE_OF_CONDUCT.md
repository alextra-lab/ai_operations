# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
as its code of conduct.

The full text of the Covenant is published at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

This Code of Conduct applies within all project spaces, and it also applies
when an individual is officially representing the project in public spaces.
This includes the GitHub repository, issues, pull requests, discussions,
and any project-affiliated communication channel.

## Reporting

Conduct concerns may be reported privately by:

- Using GitHub's **Report content** feature on the relevant comment, issue,
  or pull request.
- Emailing the maintainers (replace with the address you wish to publish
  before publishing the repo): `conduct@example.com`.

Reports are reviewed promptly and confidentially. The maintainers will
respond as outlined in the Covenant's enforcement guidelines.
