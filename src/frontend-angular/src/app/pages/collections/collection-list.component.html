info About Collections

Collections organize documents with specific embedding models for accurate semantic search.

  • Embedding Model: Each collection is bound to a specific embedding model (immutable after creation)
  • Document Assignment: Documents must be assigned to a collection during upload
  • Multi-Collection Queries: Use Cases can search across multiple collections
  • System Managed: Default collections cannot be deleted
